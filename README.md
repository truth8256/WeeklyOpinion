# Weekly Opinion Data Presentation

대통령 직무 수행 평가 및 정당 지지율 추이를 시각화하여 보여주는 데이터 프레젠테이션 프로젝트입니다. 영상 촬영 및 방송용으로 최적화된 큰 폰트와 깔끔한 UI를 제공합니다.

## 주요 기능
- **주간 추이**: 한국갤럽 및 리얼미터 조사 결과 비교
- **월간 통합 분석**: 월별 데이터 흐름 시각화
- **세부 지표**: 성별, 연령별, 지역별 세부 지지도 분석
- **데이터 연동**: CSV 파일을 통해 데이터 관리 용이

## 실행 방법
1. 저장소를 클론합니다.
2. `index.html` 파일을 웹 브라우저(Chrome 권장)에서 실행합니다.
3. 키보드 방향키(←, →) 또는 화면의 화살표 버튼으로 슬라이드를 넘길 수 있습니다.

## 데이터 수정
`data/` 폴더 내의 CSV 파일을 수정하면 차트에 자동 반영됩니다.
- `monthly_main_trends.csv`: 월별 대통령 긍/부정 평가 및 정당 지지율
- `monthly_pres_detailed.csv`: 월별 세부 지표 (성/연령/지역 등)
- 그 외 주간 데이터 CSV 파일들 (갤럽/리얼미터)

## 기술 스택
- HTML / CSS / JavaScript
- [Chart.js](https://www.chartjs.org/) (데이터 시각화)
